Workshops


LaTeX

Tired of setting format for preparing scientific papers and documents? IRIS’15 gives you a new useful tool . LaTeX is a high-quality typesetting system; it includes features designed for the production of technical and scientific documentation. The workshop will give an idea on how to use the software. With LaTeX we have to concentrate only on the contents but not on the format.

Soft Computing Techniques & Image Processing

The workshop mainly focuses on the Neural Network for providing training for the features extracted from the digital images depending on the objects to be recognized. The concepts of Fuzzy system also will be mentioned in the workshop. The design of BPN, LVQ etc along with the feature extraction techniques using Matlab will be covered. The different Digital Image Processing techniques such as per-processing, noise removal, segmentation etc., will also be discussed.

LabVIEW

LabVIEW is a system-design platform and development environment for visual programming language . It is commonly used for data acquisition, instrument control, and industrial automation on a variety of platforms including Microsoft Windows, various versions of UNIX, Linux, and Mac OS X. With LabVIEW, you can write sophisticated programs and applications in a shorter amount of time.

Machine Design

Know more about engineering softwares particularly CREO software . Creo is a scalable, interoperable suite of product design software that delivers fast time to value. It helps teams create, analyze, view and leverage product designs downstream utilizing 2D CAD, 3D CAD, parametric & direct modeling. In this workshop you will be familiarised with engineering softwares for machine designing.
